# LojaRoupas — Landing Page

Landing page de loja de roupas casual com tema dark (preto e dourado), desenvolvida com HTML, CSS e JavaScript puro como projeto de portfólio.

![Preview]
![HTML5]
![CSS3]

---

## Funcionalidades

- Layout dark com paleta preto e dourado
- Contador de sacola funcional
- Botão de favoritar produtos
- Formulário de newsletter com feedback
- Layout responsivo (mobile, tablet, desktop)
- Hover effects e micro-interações em CSS

---

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (sem frameworks)
- Google Fonts

---


## Estrutura

```
loja-roupas/
├── index.html    # Estrutura da página
├── style.css     # Todos os estilos
├── script.js     # Interações JavaScript
└── img/          # Coloque suas imagens aqui
```

## O que aprendi com esse projeto

- Estruturação semântica com HTML5
- CSS Grid e Flexbox para layouts complexos
- Variáveis CSS para theming consistente
- Responsividade com media queries
- Micro-interações com CSS transitions
- Manipulação do DOM com JavaScript puro
